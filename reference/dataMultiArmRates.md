# Multi-Arm Dataset of Rates

A dataset containing the sample sizes and events of three groups. Use
`getDataset(dataMultiArmRates)` to create a dataset object that can be
processed by
[`getAnalysisResults()`](https://docs.rpact.org/reference/getAnalysisResults.md).

## Usage

``` r
dataMultiArmRates
```

## Format

A [`data.frame`](https://rdrr.io/r/base/data.frame.html) object.
