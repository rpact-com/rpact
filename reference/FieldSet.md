# Field Set

Basic class for field sets.

## Arguments

- ...:

  Additional args passed to print/show methods

- collapse:

  Character used to join multiple output lines (default "")

- outputType:

  Type of output to capture: "all", "print", "summary", or "json"
  (default "all")

## Value

Character scalar with markdown-formatted content

## Details

The field set implements basic functions for a set of fields.
