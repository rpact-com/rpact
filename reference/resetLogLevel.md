# Reset Log Level

Resets the `rpact` log level.

## Usage

``` r
resetLogLevel()
```

## Details

This function resets the log level of the `rpact` internal log message
system to the default value `"PROGRESS"`.

## See also

- [`getLogLevel()`](https://rpact-com.github.io/rpact/reference/getLogLevel.md)
  for getting the current log level,

- [`setLogLevel()`](https://rpact-com.github.io/rpact/reference/setLogLevel.md)
  for setting the log level.

## Examples

``` r
if (FALSE) { # \dontrun{
# reset log level to default value
resetLogLevel()
} # }
```
