# Stratified Enrichment Dataset of Rates

A dataset containing the sample sizes and events of two groups. Use
`getDataset(dataEnrichmentRatesStratified)` to create a dataset object
that can be processed by
[`getAnalysisResults()`](https://rpact-com.github.io/rpact/reference/getAnalysisResults.md).

## Usage

``` r
dataEnrichmentRatesStratified
```

## Format

A [`data.frame`](https://rdrr.io/r/base/data.frame.html) object.
