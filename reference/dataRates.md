# One-Arm Dataset of Rates

A dataset containing the sample sizes and events of one group. Use
`getDataset(dataRates)` to create a dataset object that can be processed
by
[`getAnalysisResults()`](https://docs.rpact.org/reference/getAnalysisResults.md).

## Usage

``` r
dataRates
```

## Format

A [`data.frame`](https://rdrr.io/r/base/data.frame.html) object.
