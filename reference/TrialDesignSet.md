# Class for trial design sets.

`TrialDesignSet` is a class for creating a collection of different trial
designs.

## Details

This object cannot be created directly; better use
[`getDesignSet()`](https://docs.rpact.org/reference/getDesignSet.md)
with suitable arguments to create a set of designs.

## Fields

- `designs`:

  The trial designs to be compared.

- `design`:

  The trial design.

- `variedParameters`:

  A character vector containing the names of the parameters that vary
  between designs.
