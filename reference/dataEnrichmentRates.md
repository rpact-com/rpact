# Enrichment Dataset of Rates

A dataset containing the sample sizes and events of two groups. Use
`getDataset(dataEnrichmentRates)` to create a dataset object that can be
processed by
[`getAnalysisResults()`](https://docs.rpact.org/reference/getAnalysisResults.md).

## Usage

``` r
dataEnrichmentRates
```

## Format

A [`data.frame`](https://rdrr.io/r/base/data.frame.html) object.
