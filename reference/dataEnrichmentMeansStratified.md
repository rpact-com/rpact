# Stratified Enrichment Dataset of Means

A dataset containing the sample sizes, means, and standard deviations of
two groups. Use `getDataset(dataEnrichmentMeansStratified)` to create a
dataset object that can be processed by
[`getAnalysisResults()`](https://docs.rpact.org/reference/getAnalysisResults.md).

## Usage

``` r
dataEnrichmentMeansStratified
```

## Format

A [`data.frame`](https://rdrr.io/r/base/data.frame.html) object.
