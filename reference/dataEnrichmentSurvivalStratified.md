# Stratified Enrichment Dataset of Survival Data

A dataset containing the log-rank statistics, events, and allocation
ratios of two groups. Use `getDataset(dataEnrichmentSurvivalStratified)`
to create a dataset object that can be processed by
[`getAnalysisResults()`](https://docs.rpact.org/reference/getAnalysisResults.md).

## Usage

``` r
dataEnrichmentSurvivalStratified
```

## Format

A [`data.frame`](https://rdrr.io/r/base/data.frame.html) object.
